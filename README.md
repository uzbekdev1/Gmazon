# Gmazon
.NET Development Basis Course Design

这是2016-2017学年 .NET 开发基础 & .NET 实例开发 的课程设计，一个简单版的在线书店，界面配色来自老亚马逊。

~~大三下学期还会有 .NET实例开发 这门课，所以预计6月份左右会用复杂三层架构重写Gmazon。~~[已完成]

项目基于Visual Studio 2008开发，兼容IE6。
